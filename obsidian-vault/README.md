# Obsidian Vault

Este vault pode ser aberto diretamente no aplicativo Obsidian.

Pastas iniciais:
- `Inbox/` para capturas rapidas
- `Memoria/` para memorias persistentes do agente
- `Projetos/` para contexto de projetos
- `Pessoas/` para notas de pessoas
- `Logs/` para registros e resumos

O agente usa tools de memoria para ler e escrever notas Markdown neste vault.
